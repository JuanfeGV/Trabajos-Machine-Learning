Explicación de los datos utilizados

El conjunto de datos utilizado consiste enla relación entre la edad y las horas de sueño con el nivel de glucosa.

  • Edad: años del paciente.
  • Horas de sueño: tiempo promedio de descanso.
  • Glucosa: nivel estimado de glucosa (mg/dL).

Para este ejercicio se emplea un modelo de regresión lineal simple, donde las variables independientes son la edad y las horas de sueño, y la variable dependiente es el nivel de glucosa.
